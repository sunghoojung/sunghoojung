# Profile fields

Copyable source for the generated profile card:

```text
Banner: Sunghoo at GitHub
OS: macOS 26.4
Uptime: calculated from July 30, 2006
Host: Rutgers University
IDE: Cursor
Languages.Real: English
Hobbies.Software: CV, ML, Web Apps
Hobbies.Personal: Robotics
```

The editable generator is `profile_stats.py`. The daily workflow regenerates `stats_light.svg` and `stats_dark.svg`.

The original portrait cards are intentionally not regenerated by this workflow.

For full contributed-repository statistics, add a fine-grained read-only GitHub token as the `PROFILE_STATS_TOKEN` repository secret. Without it, the workflow still updates the age, public repositories, stars, commits, and followers.
